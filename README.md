# Binance Futures Testnet Trading Bot

## Overview
This project is a simplified Python trading bot that places MARKET and LIMIT orders on the Binance Futures Testnet (USDT-M).  
It is built as a CLI application with clean structure, logging, and proper error handling.

## Features
- Place MARKET and LIMIT orders
- Supports BUY and SELL
- CLI-based input using argparse
- Input validation
- API request/response logging
- Error handling for API and network failures
- Uses Binance Futures Testnet (no real funds)
